Project

Role: Team Leader

Title: Smart Guiding Assistant for Visually Impaired using MobileNetSSD

Description: Autonomous navigation and obstacle avoidance systems are critically relevant and important for visually impaired people. Even though there has been significant amount of work on obstacle, object detection and avoidance using LIDAR surveying method and camera data, there have not been much effort on focusing to provide a lightweight, reliable and portable solution for the visually impaired. Hence this project work develops a deep learning based smart wearable guiding assistant for visually impaired. The obstacle image is captured and processed to identify the image based on different features such as shape, size etc using deep learning algorithm. The detected image is the converted into speech and the same is heard through headphone. The prototype model has developed by hardware implementation using Raspberry pi. The prototype model is tested and detected using different images with accuracy of 65% to 97%.
